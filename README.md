# IT STEP – Офіційний веб-сайт

Це навчальний проєкт, створений як сучасний веб-сайт для приватної школи **IT STEP SCHOOL**. Проєкт реалізований під час навчання фронтенд-розробки.

## 🚀 Про проєкт
Цей сайт є інформаційним лендінгом, що представляє школу, її переваги, освітню платформу **MyStat** та пропонує зручну форму для запису на консультацію. 

Основна мета проєкту - відтворити структуру реального бізнес-сайту, навчитися працювати з семантичною версткою, позиціонуванням елементів та Git.

## 🛠 Технології
Проєкт розроблений з використанням:
- **HTML5**: Семантична структура сторінок.
- **CSS3**: Створення дизайну, робота з відступами, Flexbox та позиціонуванням.
- **Git**: Система контролю версій.
- **Google Fonts**: Підключення шрифту Inter.
