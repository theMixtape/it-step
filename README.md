# IT STEP SCHOOL – Навчальний проєкт (Front-End)

Це мій **навчальний проєкт**, створений під час вивчення веб-розробки. Основна мета роботи — закріпити знання з HTML5 та CSS3 на практиці, реалізувавши повноцінний інтерфейс для освітнього закладу.

## 🎓 Про проєкт

Проєкт являє собою сучасний лендінг, розроблений "з нуля". У процесі розробки я вчився структурувати контент, працювати з медіа-активами та застосовувати стилі для створення зручного інтерфейсу.

**Основні цілі навчання:**

* Реалізація семантичної верстки.
* Опанування методів позиціонування (Flexbox, Absolute).
* Робота з формами та інтеграція графічних елементів (іконки, прапори).
* Використання системи контролю версій Git для відстеження прогресу навчання.

## 🛠 Технічний стек (Навчальний набір)

Для створення цього сайту я використовував базовий набір розробника:

* **HTML5:** Структурування інформації.
* **CSS3:** Створення адаптивного дизайну та стилізація компонентів.
* **Git/GitHub:** Організація роботи над навчальним проєктом.
* **Google Fonts:** Підключення шрифтів для покращення візуальної частини.

Під час роботи над цим навчальним проєктом я відпрацював наступні навички:

1. **Робота з макетами:** Перенесення дизайну в код.
